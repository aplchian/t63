# TextField

A textfield component consists of a Label, Input and with an optional div to display instructions.

## Usage

```
  <TextField
    name="First Name"
    value={props.fname}
    onChange={props.handleChgFName} />
  <TextField
    name="Last Name"
    value={props.lname}
    onChange={props.handleChgLName} />

```
